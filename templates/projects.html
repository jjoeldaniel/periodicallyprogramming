    Projects

    A list of projects that I've worked on in the past!

    Trends Analysis

    A collaborative project within CSUF ACM chapter that uses keyword extraction with KeyBERT to analyze user messages and cache extracted data in data structures for user requests, generating a JSON file used by GitHub actions to deliver content to ACM CSUF website.

    January '23 - Present

    Discord Bot

    A Discord bot that implements Spotify API to collect album, artist, and playlist info, allowing users to set notification phrases and using prepared statements in front-facing user interactions to prevent possible SQL injection attacks, with user settings stored in PostgreSQL and ElephantSQL.

    May '22 - Present

    Genius API Wrapper

    A project that utilizes Requests HTTP library to communicate with Genius REST API, scraping the Genius website to parse song lyrics using Beautiful Soup, and currently maintained and hosting active releases on Python Package Index (PyPi), with GitHub workflows used to push releases to PyPi.

    December '22 - Present
